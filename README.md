# WOTS — Bay B2B Alpha (Godot)

This repository contains the **Warehouse Operations & Training Simulator (WOTS)** — **Bay B2B Alpha** scaffold.

## Status
- Godot project boots successfully
- Empty `Main` scene set as startup
- Core folder structure created
- Single constants config file: `res://core/config.gd`

## Run
Open the project in Godot 4.x and press ▶ Play.

## Structure
- `res://core/*` — core simulation systems (time, events, rules, roles, scoring, scenarios)
- `res://ui` — UI scenes/scripts
- `res://data` — data assets
- `res://debug` — debug utilities
